InCollege - A Student Networking Platform
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-
InCollege is a LinkedIn-inspired application designed for college students to connect, post job opportunities, and showcase their skills. 
Developed as part of the Software Engineering course over Summer 23', the project emphasizes agile software development in scrum teams, guided by Dr. James Anderson.
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-
Development Highlights:
- Agile Practices: Structured into 5 epics with tasks managed via Jira boards. Team roles rotated weekly, covering Scrum Master, Developer, and Tester responsibilities.
- Technology Stack:
  - Programming Language: Python
  - Database: SQLite3
  - Testing: Pytest
- Final Deliverable: Fully functional platform integrating features from all 5 epics, presented in a class demo alongside .pptx presentation.
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-
Features:
- User registration and login
- Profile creation and management
- Job postings and applications
- Network building through friend requests and managing connections
- Search for students by name, university, or major
- Access to useful and important links (e.g., privacy policies, language settings)
- Guest controls for managing notifications and preferences
- View and accept/reject friend requests
- Customizable language settings
